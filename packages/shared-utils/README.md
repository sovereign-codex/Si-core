# `@sovereign-codex/shared-utils`

Shared type definitions and lightweight utility helpers that are reused across Sovereign Codex packages. This module is published as an ES module and compiled from the local TypeScript sources.

## Building

```bash
pnpm --filter @sovereign-codex/shared-utils build
```

## Contents

- `types.ts` – foundational shared interfaces and type aliases.
- `utils.ts` – small helper utilities that operate on shared types.
