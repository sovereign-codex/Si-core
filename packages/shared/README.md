# `@sovereign-intelligence/shared`

Shared type definitions and lightweight utility helpers that are reused across Sovereign Intelligence packages. This module is published as an ES module and compiled from the local TypeScript sources.

## Building

```bash
pnpm --filter @sovereign-intelligence/shared build
```

## Contents

- `types.ts` – foundational shared interfaces and type aliases.
- `utils.ts` – small helper utilities that operate on shared types.
