# @sovereign-codex/shared-utils

## Identifier

Branded string identifier shared across Sovereign Codex packages.

## PaginatedResult

Normalized structure for paginated responses throughout the platform.

## toIdentifier

Casts the provided string to an Identifier without additional validation.

## assertIdentifier

Ensures the provided string can act as an Identifier, throwing if empty.

## isoDate

Formats the provided date-like value as an ISO 8601 string.
